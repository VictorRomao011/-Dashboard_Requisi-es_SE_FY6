# Dashboard Requisições SE FY26

Dashboard interativo de requisições de compras — Siemens Energy FY26 (Ago/2025 – Mai/2026).

🔗 **Acesso público:** https://victorromao011.github.io/-Dashboard_Requisi-es_SE_FY6/

## Conteúdo
- KPIs: despesa total, requisições, pedidos, fornecedores, média mensal
- Curva S de despesa acumulada vs. volume mensal
- Composição por modelo (Material / Serviço / OI) e por status
- Top 10 fornecedores e ranking Top 5
- Tabela completa de 212 registros com busca

Tecnologia: HTML/CSS/JS estático com Chart.js.
